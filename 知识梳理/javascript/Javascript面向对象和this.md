# Javascript 面向对象和 this

## 面向对象三大特性

1. 继承：子类继承父类，子类和和父类**表现得一致**
2. 多态：子类重写父类，每个子类都可以重写父类，继承同一个父类的子类**表现得不同**
3. 封装：使用属性描述符，控制外部对对象内部属性、方法的访问。属性描述符：`private、protected、public`
